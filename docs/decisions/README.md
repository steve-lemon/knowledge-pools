# Decision Log

This directory stores architecture decision records.

Use one file per meaningful decision.

Recommended filename:

```text
YYYY-MM-DD-short-title.md
```

## Template

```text
# Decision: Title

Date: YYYY-MM-DD
Status: proposed | accepted | superseded

## Context

What situation or constraint led to this decision?

## Decision

What did we decide?

## Rationale

Why is this the preferred path?

## Alternatives

What options were considered?

## Consequences

What becomes easier, harder, or riskier because of this?

## Follow-ups

What should be revisited later?
```
