# Social Sharing Workspace

This directory tracks how Knowledge Pools can be shared publicly as the project evolves.

The goal is to turn project progress into clear posts without losing technical accuracy or overpromising. Each shareable update should connect one concrete project step to one useful idea for readers.

## Files

- [strategy.md](strategy.md): audience, tone, channels, and content principles.
- [stage-plan.md](stage-plan.md): project stages and what to publish at each step.
- [post-template.md](post-template.md): reusable format for drafting posts.
- [drafts.md](drafts.md): current draft posts ready to refine.

## Publishing Rule

Before publishing any post, check:

1. Is the claim grounded in actual repository progress?
2. Does it explain one idea clearly?
3. Does it invite useful conversation?
4. Does it avoid pretending the system is more complete than it is?
